# Demon Teacher

# Building

Building is super easy, just run the appropriate script for your platform.

## Windows

Run the `build.bat` script.

## Mac / Linux

Run the `build.sh` script.

# Running

Run `./bin/demon_teacher`

# Controls

## Right Handed

```
w          - Move forward
a          - Move left
s          - Move backwards
d          - Move right
q          - 
e          - 
left shift - 
```

## Left Handed

```
i           - Move forward
j           - Move left
k           - Move backwards
l           - Move right
o           - 
u           - 
right shift - 
```

## General

```
escape - Display menu
```
